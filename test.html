Google UploadServer CORS PoC

Tests whether Google Cloud Storage API is readable cross-origin with victim's cookies.

Prerequisite: Be logged into Google Cloud Console in this browser.

Test 1: CORS Verification (Public Bucket — No Auth Needed)

Confirms the CORS misconfiguration exists by reading a public GCS bucket.

Test 2: List YOUR GCP Projects (Requires Login)

Attempts to list your GCP projects cross-origin. If this works, cookie auth is confirmed.

Test 3: List YOUR GCS Buckets (Requires Login + Project)

Enter your GCP project ID to list its buckets cross-origin.

Test 4: List Objects in a Bucket (Requires Login)

Enter a bucket name to list its objects cross-origin.

Test 5: Download a File from GCS (Requires Login)

Enter bucket and object path to download cross-origin.
